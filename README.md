# holbertonschool-sorting_algorithms

## Description
Ce projet porte sur l'implémentation de différents algorithmes de tri en langage C (Bubble sort, Insertion sort, Selection sort, Quick sort) et l'analyse de leur complexité temporelle à l'aide de la notation Big O.

## Tasks
| File | Description | Prototype |
| --- | --- | --- |
| 0-bubble_sort.c | Trie un tableau d'entiers dans l'ordre croissant avec l'algorithme Bubble sort. | `void bubble_sort(int *array, size_t size);` |
| 0-O | Contient les complexités Big O pour Bubble sort (meilleur, moyen, pire cas). | Non spécifié |
| 1-insertion_sort_list.c | Trie une liste doublement chaînée d'entiers avec l'algorithme Insertion sort. | `void insertion_sort_list(listint_t **list);` |
| 1-O | Contient les complexités Big O pour Insertion sort (meilleur, moyen, pire cas). | Non spécifié |
| 2-selection_sort.c | Trie un tableau d'entiers dans l'ordre croissant avec l'algorithme Selection sort. | `void selection_sort(int *array, size_t size);` |
| 2-O | Contient les complexités Big O pour Selection sort (meilleur, moyen, pire cas). | Non spécifié |
| 3-quick_sort.c | Trie un tableau d'entiers dans l'ordre croissant avec l'algorithme Quick sort (Lomuto). | `void quick_sort(int *array, size_t size);` |
| 3-O | Contient les complexités Big O pour Quick sort (meilleur, moyen, pire cas). | Non spécifié |
| print_array.c | Fonction utilitaire pour afficher un tableau d'entiers. | `void print_array(const int *array, size_t size);` |
| print_list.c | Fonction utilitaire pour afficher une liste doublement chaînée. | `void print_list(const listint_t *list);` |
| sort.h | Fichier d'en-tête contenant les structures de données et les prototypes. | Non spécifié |
